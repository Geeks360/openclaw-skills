## 🎨 Meta Ad Banner Creator

**The only OpenClaw skill that creates Meta ad banners the right way.**

Every other approach has the same fatal flaw: they ask the AI to generate text inside the image. The result is always broken — garbled letters, misspelled words, overlapping lines. No matter how good your prompt is, AI image models simply cannot render readable text reliably.

This skill solves that permanently with one rule:

> **Text is NEVER generated by the AI model inside the image.**
> Instead, a two-layer method is used — the AI generates only the background,
> and text is overlaid programmatically using code.

This is how professional ad agencies actually work. And now your AI assistant works the same way.

---

### What the skill covers

**Placements & sizes** — exact pixel dimensions for every Meta format: Feed 1:1 and 4:5, Stories, Reels, Carousel, Right Column, Marketplace, and In-Stream video thumbnails.

**Safe zones** — precisely where Meta overlays its own UI (profile name, CTA button, swipe indicators), so your text never gets hidden after the ad goes live.

**Three text overlay methods with ready-to-use code:**
- HTML/CSS (recommended — pixel-perfect, no dependencies)
- Python with Pillow (for automation and batch production)
- Node.js with Sharp + SVG (for developers)

**Typography rules** — which fonts survive a 375px phone screen, minimum sizes, weight requirements, contrast ratios, and why you should never use a thin or decorative font in an ad.

**Color system** — how to pick an accent color that draws the eye to your key message, and why pure white body text is a mistake.

**Four high-converting design patterns** — with ASCII layout diagrams: Photo + Bottom Text Block, Split Screen, Bold Text on Gradient, and Stories/Reels Vertical.

**Batch production workflow** — how to produce multiple A/B variants efficiently: same message/different visuals, same visual/different copy, and multi-format adaptation from a single source image.

**The 20% text rule** — Meta no longer hard-blocks text-heavy ads, but the algorithm penalizes them with higher CPM. The skill explains exactly how to stay safe.

**Full quality checklist** — 7 checks before you hand off any banner, including mobile readability simulation at 375px width.

**Complete worked example** — from brief ("hiring psychiatrists, remote, flexible hours") to finished 1080×1350 PNG, with the exact background prompt, text overlay code, and export settings.

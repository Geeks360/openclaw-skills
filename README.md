## 🎨 Meta Ad Banner Creator

**Stop wasting hours in Canva. Let your AI create production-ready Meta ad banners for you.**

Most AI assistants fail at ad creatives because they try to generate text directly inside the image — and it always comes out garbled, misaligned, or misspelled. This skill fixes that with a professional two-layer method: AI generates the background, code renders the text. The result? Clean, readable, pixel-perfect banners every time.

### What it covers

- ✅ All Meta placements: Feed (1:1 and 4:5), Stories, Reels, Carousel, Marketplace
- ✅ Safe zone rules so your text never gets hidden by Meta UI
- ✅ The two-layer method — background image + programmatic text overlay
- ✅ Prompts for AI image generation (no garbled text, ever)
- ✅ Ready-to-use code in HTML/CSS, Python (Pillow), and Node.js (Sharp)
- ✅ Typography rules: fonts, sizes, weights, contrast ratios
- ✅ Color rules for text readability on mobile
- ✅ High-converting design patterns with layout diagrams
- ✅ Batch production workflow for A/B testing
- ✅ The 20% text rule explained — avoid Meta penalizing your ads
- ✅ Full checklist before delivery

### Who it is for

Marketers, agencies, and freelancers who run Facebook and Instagram ads and want their AI assistant to do the heavy lifting — properly.

### Main advantage

Unlike generic "make me a banner" prompts, this skill follows real production standards used by ad agencies. It knows where Meta places UI overlays, which fonts hold up on a 375px phone screen, and why you should never let an AI draw text on an image.

1. Download the `SKILL.md` file above

   ```

   ```
3. Restart OpenClaw — the skill is auto-detected and ready to use
